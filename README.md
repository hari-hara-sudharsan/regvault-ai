# MantleGuard - Smart Contract Security Extension

AI-powered security analysis and gas optimization for Solidity smart contracts.

## Features

- **🛡️ Security Audits:** Comprehensive smart contract security analysis
- **⚡ Gas Optimization:** Real-time suggestions to reduce gas costs
- **🤖 AI Copilot:** Intelligent code assistance powered by AI
- **📊 Detailed Reports:** Generate comprehensive security and optimization reports
- **💡 CodeLens Integration:** Inline actions directly in your editor
- **🔍 Hover Intelligence:** Quick insights on hover over functions

## Quick Start

1. Install the extension
2. Open any `.sol` (Solidity) file
3. See the 🛡️ MantleGuard icon in the Activity Bar
4. Right-click on your contract to access features

## Commands

- `MantleGuard: Analyze Contract` - Analyze your smart contract
- `MantleGuard: Audit Contract` - Run security audit
- `MantleGuard: Optimize Gas` - Get gas optimization suggestions
- `MantleGuard: AI Copilot` - Get AI assistance
- `MantleGuard: Generate Report` - Create comprehensive reports
- `MantleGuard: Open Dashboard` - Open the main dashboard

## Requirements

- VS Code 1.120.0 or higher
- Solidity files (.sol)

## Extension Settings

This extension contributes the following settings:

- `mantleguard.apiUrl`: Set the MantleGuard API URL (default: http://localhost:3000)
- `mantleguard.enableAutoAnalysis`: Enable automatic analysis on file save

## Usage

### 1. Analyze Contract
Right-click on any Solidity file and select "Analyze with MantleGuard"

### 2. Security Audit
Run comprehensive security checks on your smart contracts

### 3. Gas Optimization
Get real-time suggestions to reduce gas costs

### 4. AI Assistance
Ask AI about your contract through the sidebar

## Known Issues

- Requires local backend API for full functionality
- Currently supports Solidity (.sol) files only
