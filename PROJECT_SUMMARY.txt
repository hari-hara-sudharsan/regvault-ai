================================================================================
TROUBLESHOOTING
================================================================================

Problem: Extension doesn't activate
Solution: Open a .sol file

Problem: Commands not showing
Solution: Ctrl+Shift+P > "Reload Window"

Problem: Backend connection fails
Solution: Start backend: cd apps/web && npm run dev

Problem: CodeLens not appearing
Solution: Save file or run "Refresh CodeLens"

Problem: Hover not working
Solution: Run "Clear Hover Cache" command

For more help, check:
  - Output panel: View > Output > MantleGuard
  - Developer Tools: Help > Toggle Developer Tools
  - Documentation: See START_HERE.md
